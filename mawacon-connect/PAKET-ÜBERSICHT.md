## Wie funktioniert's?

### Bestellungen
1. Kunde bestellt Hardware/Tarif-Upgrade/TV-Paket in der App
2. System sendet automatisch E-Mail an Mitarbeiter
3. Mitarbeiter bearbeitet Bestellung manuell
4. Kunde erhält Bestätigung per E-Mail

### Freunde werben
1. Kunde teilt seinen persönlichen Code
2. Freund gibt Code bei Vertragsabschluss an
3. System erkennt automatisch erfolgreiche Werbung
4. Beide erhalten automatisch Gutschrift
5. Kunde erhält Reminder, wenn Gutschrift aussteht

### Aktionen
- Aktionsbild kann einfach im System ausgetauscht werden
- Keine komplexe Verwaltung nötig
