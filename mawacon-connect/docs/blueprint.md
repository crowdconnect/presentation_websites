# **App Name**: Mawacon Connect

## Core Features:

- Account Overview: View invoices, contract details, and package information directly within the app.
- Contract Management: Upgrade or modify existing internet contracts directly through the app.
- Hardware Orders: Order new hardware (e.g., routers, extenders) directly through the app.
- TV Package Management: Browse and subscribe to additional waipu.tv packages (Comfort, Perfect Plus) via the app.
- Customer Referral Program: Participate in a referral program to earn bonuses by recommending Mawacon services to friends and neighbors.
- Troubleshooting via WhatsApp: Initiate a troubleshooting request via WhatsApp for quick support.
- Automated Promotion Suggestion: Suggest personalized TV packages for customers to upgrade or change services, improving their service package based on previous decisions. An LLM tool considers viewing habits.

## Style Guidelines:

- Primary color: Deep sky blue (#42A5F5), reflecting the reliable and technologically advanced nature of fiber optics.
- Background color: Light blue (#E3F2FD), provides a clean and modern backdrop.
- Accent color: Vivid orange (#FF7043), used to highlight key actions and promotions, creating a sense of energy and excitement.
- Body and headline font: 'Inter', a grotesque-style sans-serif for a clean and modern feel.
- Use modern, minimalist icons to represent features and actions within the app.
- A card-based layout will present information in an organized and easily digestible manner.
- Subtle transitions and animations will enhance user engagement and provide feedback during interactions.
